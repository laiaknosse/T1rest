REST API для вычисления частоты встречи символов


Описание

Это REST API вычисляет частоту встречи символов в переданной строке
и возвращает результат в формате JSON, отсортированный по убыванию
количества вхождений символов.


Запуск приложения

1. Склонируйте репозиторий с приложением или скачайте его архив.
2. Перейдите в корневую папку проекта.
3. Запустите приложение(Приложение будет запущено на порту 8080).


Формат входящих параметров

Для вычисления частоты встречи символов, отправьте POST-запрос
через Git BASH.
Пример:
	curl -X POST -H "Content-Type: application/json" -d 'AsdfasSsssd' http://localhost:8080/api/analyze
	

Формат исходящих параметров

После умпешного выполнения запроса, вы получите ответ
в формате JSON содеражщий карту символов и их часоту встречи.
Пример ответа:

{"s":5,"d":2,"a":1,"A":1,"S":1,"f":1}


Обработка ошибок

Git BASH вернет статус 400 Bad Request если переданная строка пустая.


Примечания

Приложение некорректно работает с кириллицей.
